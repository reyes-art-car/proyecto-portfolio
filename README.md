# Proyecto Portfolio — MRAC

Este repositorio contiene el desarrollo de mi portafolio personal, una plataforma diseñada para presentar mi trabajo, formación y trayectoria como programadora. El objetivo es disponer de un espacio profesional que refleje mi evolución técnica y facilite el acceso a mis proyectos y vías de contacto.

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  
- Figma para diseño y prototipado. https://www.figma.com/design/vLZzFhjZV94xz32XKCNx1Q/Proyecto-Portfolio?node-id=0-1&t=Fm1nv8izbiaZsjhK-1

## Estructura del proyecto

El portafolio está organizado en distintas secciones con el fin de ofrecer una navegación clara y una presentación ordenada del contenido:

- **Inicio**: mensaje principal y presentación breve.  
- **Sobre mí**: resumen personal y profesional.  
- **Proyectos**: listado de trabajos relevantes, con descripciones y enlaces.  
- **Contacto**: formulario y métodos adicionales para comunicar conmigo.

## Objetivos del desarrollo

- Mantener una estructura de código clara y fácil de mantener.  
- Conseguir un diseño adaptable a móviles y pantallas de distintos tamaños.  
- Ofrecer una navegación intuitiva y orientada al usuario.  
- Facilitar la ampliación del contenido y la incorporación de nuevos proyectos.

## Estado del proyecto

El portafolio continúa en desarrollo y se actualizará conforme progrese mi formación y experiencia profesional.

## Mejoras previstas
